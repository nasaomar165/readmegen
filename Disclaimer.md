## 6. GitHub Actions Integration

The `--gen-workflow` feature generates a CI/CD workflow that runs readmegen automatically on your repository. By enabling this:

- Your **entire repository** will be scanned and sent to the configured cloud AI provider on every push to the default branch.
- The workflow will **commit and push** AI-generated changes back to your repository using the `GITHUB_TOKEN`.
- Unreviewed AI-generated content will be published to your repository's main branch automatically.

Ensure you fully understand the implications before enabling automated workflows, particularly on repositories containing sensitive code or in organizational/enterprise settings.
